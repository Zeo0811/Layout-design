# 十字路口排版专用插件

将 Notion / 飞书文档一键转换为微信公众号排版格式的 Chrome 扩展。

---

## 安装步骤

### 第一步：下载插件

1. 打开 [GitHub 仓库](https://github.com/Zeo0811/Layout-design)
2. 点击右上角绿色 **Code** 按钮 → **Download ZIP**
3. 解压到本地任意文件夹（比如桌面）

### 第二步：在 Chrome 中加载插件

1. 打开 Chrome，地址栏输入：
   ```
   chrome://extensions
   ```
2. 打开右上角 **开发者模式**（Developer mode）开关
3. 点击 **加载已解压的扩展程序**（Load unpacked）
4. 选择刚才解压的文件夹里的 **`chrome-extension`** 子目录
5. 插件图标出现在 Chrome 工具栏即安装成功

---

## 使用方法

1. 打开任意一篇 **飞书文档 / Wiki** 或 **Notion** 页面
2. 等待页面完全加载
3. 点击 Chrome 工具栏中的插件图标
4. 在弹出窗口中点击 **⚡ 一键转换**
   - 插件会自动滚动页面以加载所有内容（飞书虚拟滚动机制）
   - 完成后显示预览
5. 点击 **📋 完整复制** 或按段落分段复制
6. 粘贴到微信公众号编辑器即可

## 自定义样式（实验室）

- 在仓库根目录的主页（`index.html`）中新增了「自定义样式」Tab，可在线调整标题、段落、列表、图片等所有样式对应的 CSS。
- 编辑器右侧提供实时预览，保存后会将样式同步到插件，在扩展的样式下拉框中即可看到自定义主题。
- 如在本地调试，可通过 `http://localhost` 打开页面，扩展会自动注入同步脚本完成交互。

---

## 注意事项

- 飞书图片使用了鉴权机制，转换时需保持飞书页面处于登录状态
- 首次在某个页面使用时，插件会自动注入内容脚本；如遇连接失败，刷新页面后重试
- 支持飞书域名：`feishu.cn`、`larksuite.com`
- 支持 Notion 域名：`notion.so`、`notion.site`
